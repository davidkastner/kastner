Personal website.

## Install
If you would like to view the site locally for making small edits,
perform the following command sequence.
Installations only have to happen once.

```brew install ruby```
```bundle add webrick```
```gem install bundler```
```bundle install```
```bundle exec jekyll serve```
```localhost:4000```
